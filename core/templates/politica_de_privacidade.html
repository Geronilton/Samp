Política de Privacidade

Esta política explica como tratamos os dados pessoais dos usuários da plataforma SAMP.

1. Dados que coletamos

  • De empresas: nome, email, categoria, endereço, Instagram, WhatsApp.
  • De usuários: nome, email, telefone, tipo de usuário, empresa vinculada.

2. Como usamos esses dados

  • Para identificar e autenticar usuários.
  • Para permitir que empresas gerenciem seus assinantes e serviços.
  • Para comunicação entre a plataforma, empresas e assinantes.

3. Compartilhamento de dados

Não compartilhamos dados pessoais com terceiros, exceto quando exigido por lei ou necessário para a operação do sistema.

4. Direitos do usuário

Você pode solicitar atualização, correção ou exclusão dos seus dados pessoais entrando em contato conosco.

5. Segurança

Implementamos medidas técnicas e administrativas para proteger os dados pessoais contra acesso não autorizado.

Última atualização: 20 de maio de 2025
